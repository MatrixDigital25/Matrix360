SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Build fails with "Cannot find module"
Solution: rm -rf node_modules && npm install --legacy-peer-deps && npm run build

Issue: Port 3000 already in use
Solution: npm run dev -- -p 3001

Issue: Vercel deployment stuck
Solution: Check build logs, clear cache in Vercel dashboard

Issue: Domain not resolving
Solution: Wait 5-10 min for DNS, check Vercel settings

Issue: Forms not saving data
Solution: Expected for MVP - backend integration needed post-launch

Issue: Mobile menu not working
Solution: Check React state management in Header component
