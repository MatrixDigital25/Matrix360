# Matrix360 - Hybrid Intelligence Platform

Production-ready Next.js website for Matrix360 Hybrid Intelligence Operating System.

## Quick Start

```bash
npm install
npm run dev
```

Visit: http://localhost:3000

## Pages

- **Homepage** - `/` - Platform overview and value proposition
- **For Enterprises** - `/for-enterprises` - Enterprise features and pricing
- **For Consultants** - `/for-consultants` - Consultant earning model
- **Enterprise Trial** - `/enterprise-trial` - Free trial signup form
- **Consultant Onboard** - `/consultant-onboard` - 3-step consultant profile
- **Enterprise Dashboard** - `/dashboard/enterprise` - Enterprise KPIs and tracking
- **Consultant Dashboard** - `/dashboard/consultant` - Earnings and opportunities
- **Strategy Room** - `/strategy-room` - Live collaboration interface

## Tech Stack

- **Framework**: Next.js 14
- **UI**: React 18 with TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Build & Deploy

```bash
npm run build
npm run start
```

## Environment

No API keys or environment variables required for basic deployment.
